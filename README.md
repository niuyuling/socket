# socket
简单的socket服务器, 文件传输和远程命令.

    socket version 1.0, socket
    Author: aixiao aixiao@aixiao.me
    socket Usage: socket [ip add] & ["-f|-k file|pid" || "command" ] | [-?h]

        -?,-h     : this help
        -d        : daemon
        -f file   : Transfer file
        -k pid    : Child process PID
        -p,       : pid file(default: socket.pid)
